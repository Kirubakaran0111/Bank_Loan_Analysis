![](/Page_1.png)

## 📌 Project Overview

This project delivers an end-to-end **Bank Loan Portfolio & Credit Risk Analysis Solution** using **MS SQL Server, Power BI, and Machine Learning**.

The objective is to transform raw banking loan data into a structured **Decision Support System** that enables:

- Loan performance monitoring  
- Risk segmentation  
- Default probability estimation  
- Charge-off exposure analysis  
- Executive-level decision making  

---

## 🚀 The Business Problem

Financial institutions face significant capital loss due to poor credit risk assessment and loan defaults.

This project builds a **360° Credit Risk Intelligence Framework** that:

- Differentiates Good vs Bad Loans  
- Tracks Month-to-Date (MTD) & Month-over-Month (MoM) trends  
- Identifies high-risk borrowers before default  
- Quantifies exposure from charged-off loans  
- Supports proactive risk mitigation strategies

# 🛠 Technical Architecture

### 1️⃣ Data Layer – SQL Server
- Built KPI aggregation queries
- Designed MTD & PMTD comparison logic
- Implemented risk bucketing using CASE statements
- Created reusable SQL Views
- Performed BI metric validation

### 2️⃣ Predictive Layer – Machine Learning
- Implemented Random Forest Classifier
- Engineered features using:
  - DTI
  - Annual Income
  - Loan Grade
  - Interest Rate
- Generated `Default_Probability`
- Created High-Risk Watchlist (>70% probability)

### 3️⃣ Business Intelligence Layer – Power BI
- Data modeling with relationships
- DAX measures for KPIs
- What-if parameter for interest rate hike simulation
- Dynamic Row-Level Security (RLS)
- Executive-ready visuals
